Text Instructions: How It Works

  1. Download & Unzip: Download the `Secret_Folder_Manager.zip` file. Unzip it to your desired location. No installation is required!
  2. Run the Launcher: Double-click on `Launch_Secret_Manager.bat` (make sure to "Run as administrator" when prompted by Windows UAC).
  3. Set Your Password: The first time you run it, you'll be prompted to set a password. Choose a strong password and remember it! This password protects your hidden folders.
  4. Select Your Folder: Click the "Select Folder" button and choose any folder on your computer that you wish to hide or manage.
  5. Hide/Show: Enter your password in the input field and click the "Execute Action" button.
    • If the folder is visible, it will become super-hidden.
    • If the folder is hidden, it will become visible again.
    • If the folder doesn't exist at the selected path, the tool will offer to create and hide it for you.
  6. Accessing Hidden Folders: When hidden, the folder will not appear in File Explorer. To access it, you must run the `Launch_Secret_Manager.bat` again, enter your password, and click "Show Folder".

    Important: To always see hidden items in Windows File Explorer (for troubleshooting), go to "View" tab -> "Show/hide" section -> check "Hidden items". Also, in "Options" -> "View" tab, uncheck "Hide protected operating system files (Recommended)".
